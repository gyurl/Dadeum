음성인식 API 시작하기

STT OpenAPI의 사용법
1. 기능 소개
STT API는 음성 파일을 텍스트로 변환하도록 제공합니다. HTTP기반의 Restful API로 구현되었습니다.
2. 애플리케이션 등록
1. DaDeum 사이트 회원가입
2. 로그인
3. STT OpenAPI 사용하기
애플리케이션 등록이 완료되면,

1. 로그인할 때 JWT 토큰을 발급하며 STT 서비스를 이용하실 수 있습니다.

2. 사용자는 API Key 발급을 통해 외부에서도 STT 서비스를 이용하실 수 있습니다.
4. 참고 사항
업로드 파일 : 2GB, 최대 4시간 제공
